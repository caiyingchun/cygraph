cygraph.algorithms¶

Contains various graph theory algorithms.

cygraph.algorithms.articulation_points¶

Functions involving articulation points.

cygraph.algorithms.articulation_points.get_articulation_points()¶

Finds the articulation points in a graph using the lowpoint theorem.

Parameters:graph (cygraph.Graph) – An undirected graph.
Returns:The articulation points in graph.
Return type:set
Raises:NotImplementedError – graph is directed.

Examples

>>> G = cg.graph(vertices=list(range(4)))
>>> G.add_edge(0, 1)
>>> G.add_edge(1, 2)
>>> G.add_edge(2, 3)
>>> alg.get_articulation_points(G)
{1, 2}

cygraph.algorithms.components¶

Functions for identifying graph components.

cygraph.algorithms.components.get_components()¶

Gets the connected components of a graph as connected graphs.

Parameters:
  • graph (cygraph.Graph) – A graph.
  • static (bint, optional) – Whether or not the graphs in the output should be static.
Returns:

Graphs that are the components of graph.
Return type:

set
Raises:

NotImplementedError – graph is directed.

Examples

>>> G = cg.graph(vertices=list(range(4)))
>>> G.add_edge(1, 2)
>>> G.add_edge(2, 3)
>>> alg.get_components(G)

cygraph.algorithms.partitioning¶

Functions related to graph partitioning.

cygraph.algorithms.partitioning.partition_karger()¶

Partitions a graph into two graphs. Does not change the inputted graph in any way.

Parameters:
  • graph (cygraph.Graph) – A graph.
  • static (bint, optional) – Whether or not the graphs in the output should be static.
Returns:

Two graphs that are the result of a random partition of graph, as well as the cutset of the partition.
Return type:

tuple
Raises:

NotImplementedError – graph is directed.

Examples

>>> G = cg.graph(vertices=[1, 2])
>>> G.add_edge(1, 2)
>>> alg.partition_karger(G)
(<DynamicGraph; vertices=[1]; edges=set()>, <DynamicGraph; vertices=[2]; edges=set()>, {(1, 2, 1.0)})

cygraph.algorithms.shortest_path¶

Functions for finding shortest paths in graphs.

cygraph.algorithms.shortest_path.get_shortest_path_dijkstra()¶

Takes a graph and finds the shortest path between two vertices in it using dijkstra’s algorithm.

Parameters:
  • graph (cygraph.Graph) – A graph.
  • source – One of the vertices in graph.
  • target – The other vertex in graph.
Returns:

The list of vertices that constitute the shortest path between source and target.
Return type:

list

Examples

>>> G = cg.graph(directed=True, vertices=list(range(3)))
>>> G.add_edge(0, 1)
>>> G.add_edge(1, 2)
>>> G.add_edge(2, 0)
>>> alg.get_shortest_path_dijkstra(G, 2, 1)
[2, 0, 1]

>>> G = cg.graph(vertices=list(range(3)))
>>> G.add_edge(0, 1)
>>> G.add_edge(1, 2)
>>> G.add_edge(2, 0)
>>> alg.get_shortest_path_dijkstra(G, 1, 2)
[1, 2]
