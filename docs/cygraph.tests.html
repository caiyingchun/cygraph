cygraph.tests¶

Unit tests for Cygraph. They require the installation of pytest and can be run with the command: .. code_block:: bash

pytest

cygraph.tests.test_algorithms¶

Unit tests for algorithms implemented in cygraph/algorithms.pyx

cygraph.tests.test_algorithms.test_get_articulation_points()¶

Tests get_articulation_points function.

cygraph.tests.test_algorithms.test_get_components()¶

Tests get_components and get_number_components functions.

cygraph.tests.test_algorithms.test_get_min_spanning_tree()¶

Tests get_min_spanning_tree function.

cygraph.tests.test_algorithms.test_get_shortest_path_dijkstra()¶

Tests get_shortest_path_dijkstra function.

cygraph.tests.test_algorithms.test_get_strongly_connected_components()¶

Tests get_strongly_connected_components and get_number_strongly_connected_components function.

cygraph.tests.test_algorithms.test_partition_karger()¶

Tests partition_karger function.

cygraph.tests.test_graph¶

Unit tests for classes implemented in cygraph/graph.pyx

cygraph.tests.test_graph.test_attributes()¶

Tests various edge and vertex attribute-related methods.

Tests:
  • set_vertex_attribute
  • get_vertex_attribute
  • set_edge_attribute
  • get_edge_attribute

for StaticGraph and DynamicGraph classes

cygraph.tests.test_graph.test_constructor()¶

Tests initialization of a StaticGraph and DynamicGraph objects.

cygraph.tests.test_graph.test_edges()¶

Tests various edge-related methods.

Tests:
  • add_edge
  • get_children
  • get_parents
  • edges
  • has_edge
  • remove_edge

for StaticGraph and DynamicGraph classes.

cygraph.tests.test_graph.test_vertices()¶

Tests various vertex-related methods.

Tests methods:
  • add_vertex
  • has_vertex
  • vertices
  • remove_vertex

for StaticGraph and DynamicGraph classes.
